xlNode_java

This is a Java implementation of the XLattice node. XLattice is a secure peer-to-peer communications infrastructure.

Projecst Status

This is the 2006 Sourceforge implementation adapted to make its unit tests succeed under Java 8. The code is stable and all unit tests pass.

While the Java code is stable in its current form, it is not yet interoperable with the Go and Python3 implementations of the code.
